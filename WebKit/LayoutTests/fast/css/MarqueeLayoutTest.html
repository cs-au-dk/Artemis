Tests: the height attribute for the marquee tag

Conditions:
Because marquee is a WinIE element, we are honoring their treatment of the heigh attribute by making sure it is overridden when the given height is smaller than the font height. We also make sure that the default height of a vertical marquee is 200px.

If successful, in the horizontal group, the pink marquee should be large, and the others should all be the same size. None of the text should be cut off. In the vertical group, the aqua marquee should be 200px, and the others should be 100px.
