Chrome Canopy

..is a fractal zoomer. Press play to start falling. Click to go faster. You'll need a web browser with a fast JavaScript engine like Google Chrome, Safari or Firefox.

  • space / p — play / pause
  • n — make a new tree
  • m — toggle mutuation
  • b — toggle bloom
